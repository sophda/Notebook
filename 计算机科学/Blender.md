## 快捷键

```
/  聚焦模式
Tab 切换物体模式or编辑模式
E 基础，在当前面的基础上，不改变当前面
G 修改高度，修改当前面
S 缩放，当前面
1 2 3 切换点线面
小键盘上的123则是三视图切换
ALT+E 沿着法向挤出
ALT+Z 切换透视模式
shift+a 添加物体

F 封口


全选物体——A
框选物体——B
反选物体——Ctrl+I
删除物体——X（可以直接单手操作好嘛，左手键盘，右手鼠标，不至于一直放开右手去Delete）
```

```

```
